# 🔐 Vault Key — Password Generator

A sleek, dark-themed password generator built with React. Features a glassmorphic 3D UI, real-time strength analysis, and fine-grained character set controls.

## ✨ Features

- **Real-time password generation** — updates instantly as you adjust settings  
- **Live strength meter** — 6-tier scoring from *Weak* to *Fortress*  
- **4 character set toggles** — Uppercase, Lowercase, Numbers, Symbols  
- **Guaranteed character inclusion** — at least one character from each enabled set  
- **Length control** — slider from 6 to 64 characters  
- **One-click copy** — clipboard support with a visual confirmation flash  
- **Regenerate button** — generate a new password without changing settings  
- **Dark glassmorphic UI** — aurora glow, dot-grid texture, 3D icon perspective

## 🚀 Getting Started

### Prerequisites

- Node.js 16+  
- npm or yarn  

### Installation

```bash
# Clone the repository
git clone https://github.com/your-username/vault-key.git
cd vault-key

# Install dependencies
npm install

# Start the development server
npm run dev
